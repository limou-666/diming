# Companion AI Demo

uni-app + Vue 3 的跨平台对话类应用前端演示工程。

## Scripts

- `npm install`
- `npm run dev:h5`
- `npm run build:h5`
- `npm run dev:mp-weixin`
- `npm run build:mp-weixin`

## Pages

- `pages/sessions/index` 会话列表
- `pages/chat/index` 聊天详情
- `pages/contact/index` 联系人详情
- `pages/profile/index` 个人中心

## Notes

- 全部数据来自 `src/mock/data/*.json`
- 图片消息素材位于 `src/static/mock`
- 当前已验证 `npm run build:h5` 可通过
